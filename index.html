功能特點
  • 自動以指定的間隔重新載入 HTML 檔案或網頁。
  • 使用簡單的介面載入和查看 HTML 內容。
  • 能夠提取並顯示載入網頁的 HTML 代碼。
  • 具備 WebView2 控件相容性，提供更強大的網頁瀏覽功能。
  • 開放原始碼且允許任何用戶編輯
